# Projeto 01 - Sistema de Gerenciamento de Farmácia"

 

Este é o primeiro projeto do curso de nivelamento em Front-End, focado em construir uma base sólida com HTML5 semântico e CSS3 moderno.


 

## 🚀 Sobre o Projeto

 

O projeto consiste em uma página de links centralizada, similar ao Linktree, que serve como um portfólio inicial simples e um cartão de visitas digital. A página é totalmente responsiva e foi construída com as melhores práticas de acessibilidade e organização
    de código.

 

## 🛠️ Tecnologias Utilizadas

 

- **HTML5:** Utilização de tags semânticas (`<main>`, `<header>`, `<section>`, `<footer>`) para uma estrutura clara e otimizada para SEO.

- **CSS3:** Estilização moderna com:

  - Flexbox para layout centralizado.

  - Variáveis CSS (Custom Properties) para um tema fácil de manter.

  - Design Responsivo para adaptação em desktops e dispositivos móveis.

  - Boas práticas de acessibilidade (atributos `alt` e classes para leitores de tela).

 

## 💻 Como Executar

 

1. Clone este repositório.

2. Abra o arquivo `index.html` em seu navegador de preferência.


 

Não há necessidade de instalação de dependências.﻿﻿
